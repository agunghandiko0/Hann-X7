# 🚀 HANN X7 - WHATSAPP WARMING BOT

Bot AI otomatis untuk memanaskan akun WhatsApp agar reputasi akun meningkat dan tahan banned. Menggunakan interaksi **Human-Like AI**.

---

## ✨ Fitur Utama
* 🤖 **AI Humanized:** Chat natural, gaul, & simulasi typo.
* 🔥 **Auto-Warming:** Bot saling balas chat otomatis antar sesi.
* 🛡️ **Anti-Ban:** Delay random & simulasi mengetik.
* 🔑 **HWID Lock:** Aman dengan sistem lisensi per perangkat.
* 🔄 **Easy Update:** Update script via `git pull`.
